# Preeyansh Arora | Manufacturing Engineering Portfolio

A React-based portfolio application with a "Mechanical Arcade" aesthetic, combining engineering precision with 80s retro gaming elements.

## Features
- **Pac-Man Theme**: Custom CSS animations and 8-bit typography.
- **Mechanical UI**: Tactile card and button designs with hard shadows.
- **Dark/Light Mode**: Full theme support with high-contrast color palettes.
- **Custom Cursor**: A pixelated, interactive cursor for a unique browsing experience.
- **Project Deep-Dives**: Detailed views for engineering projects with technical specs.

## Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`

## Tech Stack
- React 18+
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- TypeScript
